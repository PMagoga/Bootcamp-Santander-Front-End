<p> — Parágrafo

A tag <p> é usada para criar parágrafos de texto.

Formatando Texto

A tag <mark> destaca o texto com um fundo amarelo.
A tag <small> diminui o tamanho da fonte.
A tag <i> coloca o texto em itálico.
A tag <u> sublinha o texto: exemplo de sublinhado.
A tag <strong> deixa o texto em negrito.

<ol>, <ul> e <li> — Listas

Existem dois tipos principais de listas:

<ol> — Lista Ordenada

Cria uma lista numerada com itens (<li>):

  1. Item 1
  2. Item 2
  3. Item 3

<ul> — Lista Não Ordenada

Cria uma lista com marcadores (bolinhas):

  • Maçã
  • Banana
  • Laranja

<a> — Links

A tag <a> cria um link para outro site. Exemplo:

Visite o MDN Web Docs para aprender mais sobre HTML.

<sub> e <sup> — Subscrito e Sobrescrito

Usadas para fórmulas matemáticas ou químicas:

Água: H2O (usando <sub>)

Potência: 210 = 1024 (usando <sup>)

<blockquote> — Citação em Bloco

"A simplicidade é o último grau de sofisticação." – Leonardo da Vinci

Essa tag é usada para exibir uma citação longa em destaque.

<hr> — Linha Horizontal

Usada para separar seções de conteúdo com uma linha horizontal, como as linhas que você viu ao longo desta página.
